# Sudoku

A clean, ad-free Sudoku playable directly in the browser.

---

## How to Play

| Action | How |
|--------|-----|
| Select a cell | Click it |
| Enter a number | Click numpad or press 1–9 |
| Erase a cell | Backspace / Delete or click ⌫ |
| Move between cells | Arrow keys |
| Pencil mode | Click ✏️ Pencil button |
| Get a hint | Click 💡 Hint, then click any cell |
| New game | Click ↺ New Game |
| View history | Click 📋 History |

## Play
Open `index.html` in any browser — no install needed.
